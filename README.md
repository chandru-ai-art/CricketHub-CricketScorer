# CricketHub – Mobile Offline Cricket Scoring App

CricketHub is a professional mobile-first offline cricket scoring application built using HTML5, CSS3, and Vanilla JavaScript (ES6+). It functions as a Progressive Web App (PWA) allowing users to score matches ball-by-ball completely offline.

## Features
- **Mobile-first UI:** Portrait orientation with an Android-style material interface.
- **Offline Capability:** Works completely offline after installation as a PWA via Service Workers.
- **Match Setup:** Customize team names, toss details, and player rosters.
- **Live Scoring:** Ball-by-ball scoring including runs, extras (wides, no balls, leg byes), and wickets.
- **Detailed Scorecards:** View live batting and bowling statistics.
- **Match History:** View past completed matches.
- **Statistics:** Overall insights like total matches, run rate, etc.

## Technology Stack
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript (ES6+)
- IndexedDB (for offline data storage)
- Service Worker (for PWA capabilities)

## Installation & Setup
1. Clone the repository.
2. Serve the directory using any static web server.
3. Open in a modern web browser.
4. (Optional) Install the app as a PWA on your mobile device for the full offline experience.
